# Obsidian HWP Converter

옵시디언(Obsidian)의 마크다운(.md) 노트를 파이썬(Python)과 한컴오피스 오토메이션을 이용하여 **한글(.hwp)** 파일로 변환해주는 플러그인입니다.

## 주요 기능 (Features)
*   **현재 노트 변환**: 보고 있는 노트를 즉시 HWP 파일로 내보냅니다.
*   **일괄 변환 (Batch Export)**: 폴더 내의 모든 노트를 한 번에 HWP로 변환합니다.
*   **들여쓰기 옵션**: 문단 시작 부분에 자동으로 띄어쓰기(공백)를 추가할 수 있습니다.
*   **폴더 선택**: 저장할 경로를 직접 지정하거나 선택할 수 있습니다.

## 사전 요구 사항 (Requirements)
이 플러그인은 **Windows** 환경에서 **한컴오피스(한글)**가 설치되어 있어야 작동합니다.

1.  **Windows OS** (Mac/Linux 미지원)
2.  **한컴오피스** (한글 2010 이상 권장)
3.  **Python** 설치
4.  **pywin32** 라이브러리 설치
    ```bash
    pip install pywin32
    ```

## 설치 방법 (Installation)
1.  이 저장소를 다운로드하거나 클론합니다.
2.  `manifest.json`, `main.js`, `styles.css`, `scripts/` 폴더를 옵시디언 볼트의 `.obsidian/plugins/obsidian-hwp-converter/` 경로에 복사합니다.
3.  옵시디언 설정 > 커뮤니티 플러그인 > **HWP Converter**를 활성화합니다.

## 사용 방법 (Usage)
### 단일 파일 변환
1.  변환할 노트를 엽니다.
2.  명령어 팔레트(`Ctrl/Cmd + P`)에서 `HWP Converter: 한글(HWP)로 변환`을 실행합니다.
3.  또는 탐색기에서 파일 우클릭 > `한글(HWP)로 내보내기`를 선택합니다.
4.  파일명과 저장 경로를 확인하고 **내보내기**를 클릭합니다.

### 폴더 일괄 변환
1.  탐색기에서 변환할 폴더를 우클릭합니다.
2.  `한글(HWP)로 폴더 내보내기`를 선택합니다.
3.  저장 경로를 확인하고 **일괄 내보내기**를 클릭합니다.
    *   *팁: 첫 번째 파일 변환 시 뜨는 보안 경고창에서 **[모두 허용]**을 누르면 이후 파일들은 자동으로 처리됩니다.*

## 주의 사항 (Troubleshooting)
*   **보안 경고창**: 변환 시 한글(Hwp) 프로그램의 보안 탐색기가 뜰 수 있습니다. 이는 외부 스크립트가 한글을 제어할 때 발생하는 정상적인 절차입니다. **[접근 허용]** 또는 **[모두 허용]**을 눌러주세요.
*   **변환 중 조작 금지**: 변환이 진행되는 동안에는 마우스나 키보드 조작을 최소화해 주세요. 한글 창의 포커스가 바뀌면 오류가 발생할 수 있습니다.
